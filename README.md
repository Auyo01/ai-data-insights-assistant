# 📊 AI Data Insights Assistant

AI Data Insights Assistant is a web application that helps users analyze datasets using AI-powered insights, statistical summaries, visualizations, and correlation analysis.

## 🚀 Live Demo

https://ai-data-insights-assistant-3teuh5r5esrrwyezwghcye.streamlit.app/

## 🔗 GitHub Repository

https://github.com/Auyo01/ai-data-insights-assistant

## ✨ Features

- Upload CSV datasets
- AI-powered dataset analysis using Llama 3
- Statistical summaries
- Missing value detection
- Correlation analysis
- Interactive visualizations
- Downloadable summary reports
- Dataset preview and exploration

## 🤖 AI Capabilities

The application uses Groq and Llama 3 to generate:

- Key observations
- Data quality insights
- Trend analysis
- Actionable recommendations

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Auyo01/ai-data-insights-assistant.git
```

Navigate to the project:

```bash
cd ai-data-insights-assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
